Owning the clean.sh script & Inserting Reverse Shell Script


Upon closer inspection at the script, the code on line seven is the most interesting to me. This is the main function of the script that is responsible for saving the output to removed_files.log. I can insert my own reverse shell code at this line to obtain a reverse shell connection.

Privilege Escalation
